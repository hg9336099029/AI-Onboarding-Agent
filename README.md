## Overview

This project provides an AI-powered assistant that helps developers onboard to new codebases by:
- Analyzing code structure and relationships
- Answering natural language questions about the codebase
- Providing intelligent code navigation and explanations

## Tech Stack

### Backend
- **Framework**: FastAPI
- **Language**: Python
- **AI/ML**: LangChain, OpenAI/Gemini APIs
- **Database**: Vector database for embeddings

### Frontend
- **Framework**: React
- **Language**: JavaScript/TypeScript
